>## 字符

字符串的组成单位是字符。

* Unicode编码：可以有单字节编码、双字节编码和四字节编码，它们的表现形式是`\u{n}`,其中n为1~8个十六进制数
**提示：** 在swift语言中，不能使用单引号方式，必须使用双引号（“”）把字符括起来。字符串也是使用双引号（“”）把字符括起来。字符类型是:Character，如果省略，编译器自动推断出的类型不是字符类型，而是字符串类型。
* 转义符
```
\t : 水平制表符tab
\n : 换行
\r : 回车
\" : 双引号
\' : 单引号
\\ : 反斜线
```
